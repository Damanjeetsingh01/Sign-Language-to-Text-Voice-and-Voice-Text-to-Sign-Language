# Sign Language to Text & Voice and Text/Voice to Sign Language

## 📌 Project Overview

This project is a bidirectional communication system designed to bridge the communication gap between hearing/speech-impaired individuals and normal users. The system supports:

* **Voice/Text to Sign Language Conversion**
* **Sign Language to Text & Voice Conversion**

The application uses Machine Learning, Computer Vision, and Speech Recognition techniques to provide real-time communication.

---

# 🎯 Features

## 🔊 Speech to Sign Language

* Accepts voice input from the user
* Converts speech into text
* Displays corresponding sign language gestures/images

## ✋ Sign Language to Text & Voice

* Captures hand gestures using a webcam
* Recognizes gestures using a trained ML model
* Converts gestures into text
* Generates voice output from recognized text

## 🖥 User-Friendly Interface

* Simple GUI developed using Tkinter
* Easy navigation between modules
* Real-time output display

# 🚀 Installation & Setup

## Step 1: Clone the Repository

```bash
git clone <repository-link>
```

## Step 2: Navigate to Project Folder

```bash
cd Audio-Speech-To-Sign-Language-Converter-master
```

## Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

If requirements file is unavailable:

```bash
pip install django opencv-python numpy tensorflow keras nltk SpeechRecognition pyaudio
```
